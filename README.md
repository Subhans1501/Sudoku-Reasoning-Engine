# # Sudoku Logic Engine: Automated Reasoning
### # Solving Sudoku via Propositional and First-Order Logic (FOL)

This project explores the power of formal logic in solving times combinatorial puzzles. By reducing Sudoku rules to logical entailment problems, this engine uses resolution and inference to derive consistent solutions.
